# Security Policy

## Official distribution

Official WaveSpeed Desktop and WaveSpeed Mobile builds are distributed only from:

- https://github.com/WaveSpeedAI/wavespeed-desktop/releases
- links published on https://wavespeed.ai

Treat packages from any other repository, domain, app-store publisher, or download
service as unofficial unless one of the official locations above explicitly links to it.

## Reporting

Please use GitHub's private vulnerability-reporting feature for security issues when it
is available. For suspected impersonation, malicious redistribution, or a compromised
download, open an issue containing only public evidence and do not post credentials,
private keys, personal data, or exploit details.

Include the download URL, file name, SHA-256 digest, application version, platform, and
any observed signing identity. WaveSpeedAI may request the file through a private
channel for further analysis.
